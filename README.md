# LXMF_Bot
Simple Message and Response bot for the LXMF network, commands are listed in a seperate file. LXMFBot.py is a generic handler.


# Plugins
This bot allows the use of plugins to expand capabilities and keep the code clean

# Admin Commands
This bot keep some commands behind admin password or whitelisted lxmf addresses

# Network diagnostics
Commands for network diagnostics are presetup

# Commands cooldown
Rate limits for each user to limit the spamming of commands, and to limit repeats of the same command.
